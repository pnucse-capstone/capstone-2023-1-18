There was no significant difference in accuracy when the model was configured without any seasonal distinction and when the model was configured separately by season. The reason for this is that POI was selected as a tourist destination or restaurant with a high frequency of visits regardless of the season, rather than whether or not POI visits according to the season. In addition, the most predictive performance was when the number of points of interest (POIs) of users was selected as 25, which is thought to be due to the fact that the smaller the number of clusters, the larger the number of GPS information contained in one cluster. The data provided by the Jeju data hub was not disclosed, so the only information available was the date and time when the user's log records were collected, and the latitude and longitude data of the place of visit. Therefore, since the model could not be constructed considering the characteristics of the user, there is a limitation that the only information used to predict the next visit location of the traveler is the current visit location. In addition, because the system used a probability model called HMM, there were cases where the next expected movement point of some users was far from the current location. It could not be applied due to the limitation of the given time, but in the case described above, it is necessary to apply a method to correct it.

    What is HMM?

    통계적 마르코프 모형의 하나로, 시스템이 은닉된 상태와 관찰가능한 결과의 두 가지 요소로 이루어졌다고 보는 모델

    As one of statistical Martov model, it is a model that the system that system was made of two elements that are made.

    Hidden State

    시간에 대하여 은닉 변수(hidden variable) x(t)의 값이 주어질 때, 특정 시간 t t에서의 은닉 변수 x(t)의 조건부 확률 분포는 오직 은닉 변수 x(t-1)에만 의존하는 것을 관찰할 수 있다.
    그 이전의 시간들 (< t-1)은 x(t)에 영향을 주지 않는데, 이를 마르코프 특성이라 한다. 이와 비슷하게 시간 t에서 관측된 변수 y(t)의 값은 오직 은닉 변수 x(t)에만 영향을 받는다.

    Transition

    은닉 상태 공간은 범주 분포로 모델링 될 수 있는 N개의 가능한 상태들 중 하나로 구성되어 있다고 가정된다. 각각의 시간 t에서 은닉 변수는 총 N개의 가능한 상태들을 가질 수 있고, 시간 t+1에는 은닉 변수가 N개의 가능한 상태로 전이할 확률이 존재한다.
    따라서 이러한 확률은 N 2 개의 전이 확률로써 표현될 수 있다. 이 때 한 상태의 전이 확률의 합은 항상 1이 되어야 한다. 따라서 N X N 전이 확률 행렬은 마르코프 행렬이다. 다른 전이 확률들을 모두 알고 있을 때 마지막 전이 확률은 당연하게 도출될 수 있기 때문에 전체 전이 모수(transition parameter)의 수는 N(N-1)이다.

    Emission

    각 N개의 가능한 상태들에 대하여 특정 시간에서의 은닉 변수들이 주어졌을 때 관측 변수들의 분포를 제어하는 출력 확률들의 집합이 존재한다. 이 집합의 크기는 관측 변수들의 특성에 의해 결정된다. 예를 들어, 만약 관측 변수가 범주 분포의 영향을 받고 M개의 가능한 값을 가질 경우 M-1개의 독립된 모수들이 존재함을 알 수 있다. 따라서 모든 은닉 상태들에 대해 전체 출력 모수들의 개수는 N(M-1)개가 된다. 반면에 만약 관측 변수가 임의의 다변량 정규분포를 따르는 M차원 벡터일 경우 평균에 영향을 주는 M개의 모수와 공분산 행렬에 영향을 주는 M ( M + 1 ) 2 개의 모수가 존재하므로 전체 N ( M + M ( M + 1 ) 2 ) = N M ( M + 3 ) 2 = O ( N M 2 ) 개의 출력 모수가 존재하게 된다. 이런 경우에는 M M 값이 작지 않다면 관측 벡터의 각 원소들을 서로 독립으로 가정하거나, 서로 독립이면서 고정된 수의 인접 원소들을 가진다는 가정 등을 통해 공분산의 특징을 제한하는 것이 실용적이다.

    초기 확률 : 각 사용자 경로의 시작 지점에서 POIi가 관측된 횟수를 모든 관심 지점(POI)에 대하여 구한다. 그리고 이를 전체 사용자의 수로 나누어 정의
    전이 확률 : 은닉 상태(POI)에서 어떤 은닉 상태(POI)로 이동할 확률을 의미한다. 이를 도출하기 위하여 사용자 경로에서 은닉 상태의 전이가 일어나는 횟수를 구하고 이를 전체 상태 전이 횟수로 나누어 준다.
    방출 확률 : 특정 은닉 상태(qi)일 때 어떤 관측값(Observation ID)가 관측된 횟수를 전체 관측 횟수로 나누어 구한다

    Start Probability : The number of times POIi is observed at the starting point of each user path is obtained for all points of interest (POI). And defined by dividing it by the total number of users.
    Transition Probability : It refers to the probability of moving from the hidden state (POI) to a certain hidden state (POI). To derive this, the number of times the hidden state transition occurs in the user path is obtained and divided by the total number of state transitions.     Emissiong Probability : In the case of a specific hidden state (qi), the number of observations (Observation ID) is obtained by dividing the number of observations by the total number of observations.
